inlinemultiselect

A plugin that converts multiple <select> into editable, inline lists.

Download this project as a .zip file Download this project as a tar.gz file

This jQuery plugin is based on original code to create checkbox lists from multiple select elements by Kae Verens at:

  • http://verens.com/archives/2005/04/27/son-of-multiselect/

The plugin was originally hosted on Google code, and had been left there for some time without modification. Thanks to Kae Verens for his contributions to the plugin, both in the original code and in his modifications and suggestions for enhancements. Thanks also to goodladybird for identifying a bug and making me aware of internationalisation issues.

Please make sure you read the Options page and Usage page in the project wiki, and submit any issues you find to the Issue tracker.

Adding a callback to the click/change event

You can add a callback function to the click/change event of a checkbox to perform additional processing to either the checkboxes in the group, or use the values from the checkboxes in the group elsewhere in the page. The function takes a single argument which is an array containing objects representing all the checkboxes in the group. Each of these objects has the following properties set:

id
auto-generated ID of the new checkbox element
title
originally the text of the <option> element
value
originally the value of the <option> element
name
originally, the name of the multiple <select>
checked
boolean value representing the checked state of the checkbox

The callback function will be executed in the context of the checkbox whose click/change event has been triggered, so this will refer to the checkbox element.
